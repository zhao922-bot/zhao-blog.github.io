大家好！今天我想分享一下我在 GitHub 上的一些项目。这些项目涵盖了桌面应用、移动应用和浏览器扩展，主要围绕阅读和翻译工具展开。

📚 微信读书伴侣 (WeRead Companion)

Flutter 版本 (v2.1.0)

这是微信读书的移动伴侣工具，使用 Flutter 开发，支持 Android 平台。

主要功能：

  • 我的书架 - 网格展示书籍封面，支持搜索和状态筛选
  • 笔记管理 - 浏览、搜索、收藏所有笔记和划线
  • 金句卡片 - 将划线生成精美卡片，支持导出为高清 PNG 图片
  • 阅读报告 - 阅读数据可视化和统计
  • AI 摘要 - 使用 AI 为书籍生成摘要

技术栈：Flutter 3.x + Dart 3.x、Material Design 3、sqflite、dio、fl_chart、provider

PyQt6 版本 (v2.0.1)

这是微信读书的桌面伴侣工具，使用 Python + PyQt6 开发。

主要功能：

  • 数据同步 - 一键同步微信读书的书架、划线和笔记
  • 我的书架 - 网格展示书籍封面，支持搜索和状态筛选
  • 笔记管理 - 浏览、搜索、收藏、导出所有笔记
  • 金句卡片 - 将喜欢的句子生成精美图片
  • 阅读报告 - 阅读数据可视化和统计

技术栈：Python 3.10+、PyQt6、SQLite、PIL/Pillow、Matplotlib、Selenium

🌐 自定义 LLM 翻译扩展

这是一个隐私优先、可自定义的浏览器扩展，使用任何 OpenAI 兼容 API 翻译网页内容。

主要特性：

  • 通用 API 支持 - 支持 OpenAI、DeepSeek、Ollama、vLLM、Azure 等
  • 多目标翻译 - 同时翻译成多种语言
  • SPA 优化 - 自动检测并翻译动态内容
  • 隐私优先 - API 密钥本地存储，数据不离开设备
  • 可自定义提示词 - 定义自己的 AI 翻译系统指令

技术栈：JavaScript、HTML、CSS，MIT 许可证

📖 EPUB 整书翻译工具

这是一个使用自定义 LLM API 翻译 EPUB 电子书的工具，保留原始排版结构。

主要特性：

  • 保留排版 - HTML 标签、图片、目录、样式全部保留
  • 逐块翻译 - 每个段落独立翻译，缓存精确匹配
  • 断点续传 - 中断后可继续，已翻译内容不重复请求
  • 双语对照 - 可选择原文+译文对照显示
  • 进度显示 - 实时翻译进度和剩余时间

支持的 API：DeepSeek、OpenAI、通义千问、智谱 GLM、Ollama、vLLM 等
